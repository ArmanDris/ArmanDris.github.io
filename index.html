Dino Match

Dino Match is an image similarity search engine that uses an image embedder and a vector similarity search engine. The script first creates embeddings for each image in its dataset, it then uploads them to QDrant. When a user selects an image QDrant's similarity search endpoint is called to return semantically similar images.
